# ABDQuiz

Sistema de entrenamiento de exámenes de alto rendimiento, construido con Next.js 16, React 19 y MongoDB.

## 🚀 Inicio Rápido

1. Navega a la carpeta web: `cd web`
2. Instala dependencias: `pnpm install`
3. Inicia el servidor: `pnpm dev`

## 🏗️ Arquitectura
- **Frontend/Backend:** Next.js 16 (App Router) en `web/src/`.
- **Base de Datos:** MongoDB Atlas.
- **Estilos:** Tailwind CSS 4 + Shadcn UI.
